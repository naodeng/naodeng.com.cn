# Source Priority

Use these sources in descending trust order when building term explanations:

1. Standards and glossary organizations
- ISTQB Glossary
- ISO/IEC/IEEE standards overviews

2. Official engineering organizations
- Google Testing Blog
- Google SRE docs
- Martin Fowler

3. Established QA engineering vendors
- Atlassian
- BrowserStack
- Applitools

4. Community technical sources
- Medium
- Engineering blogs with concrete examples and reproducible details

5. Encyclopedia and broad references
- Wikipedia
- Baidu Baike

## Acceptance Criteria per captured item

- Has direct relevance to the term (or clearly marked adjacent concept)
- Contains explanatory detail (definition, method, examples, metrics, or pitfalls)
- Has stable URL
- Includes one-line evidence note
