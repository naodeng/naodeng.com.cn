新技术分享：Mac OS 下 edge 浏览器开启新 UI

Feb 15, 2023 · nao.deng
  • 打开 edge 浏览器
  • 在地址栏输入命令 edge://flags/
  • 在 flags 的页面输入 11 进行搜索
  • 在搜索结果下选择“Show Windows 11 visual effects in title bar and toolbar”将状态变更为启用

  • 重启浏览器，即可看到新的 edge 浏览器 UI
