# UI Redesign PRD v2

## 目标
- 在不改 Header/Footer 的前提下，提升站点现代感、专业感与可读性。
- 保持双语一致性（`en` / `zh-cn`）与内容结构稳定。
- 提升首页转化（点击文章/词条/导航）与长文阅读完成率。

## 范围
- 首页、博客详情、Wiki 首页/详情、Guild 首页/详情。
- 设计令牌、排版系统、TOC、代码块、阅读体验组件。

## 非范围
- 内容重写、信息架构大改、新增 JS 框架。

## 关键需求
1. 统一 Design Tokens（颜色、阴影、圆角、字体、spacing、motion）。
2. 文章页可读性优化（68ch、标题层级、meta 信息、阅读时间）。
3. 代码块增强（语言标识、复制按钮、无 API 优雅降级）。
4. 阅读进度条 + 回到顶部。
5. Wiki/Guild 卡片化与筛选交互优化。
6. TOC 强化（h2/h3、active 高亮、移动端折叠）。
7. 响应式和可访问性达标（focus、skip link、reduced motion）。

## 成功指标
- 首页关键区块 CTR 提升 >= 15%。
- 文章页平均滚动深度提升 >= 20%。
- CLS/LCP 不退化，移动端 Lighthouse 可访问性 >= 95。
- UI 回归缺陷（P1/P2）<= 2 个/发布周期。

## 约束
- 技术栈：Astro + vanilla JS。
- 兼容：现代浏览器 + 移动端。
- 性能预算：首屏 JS 不新增重量依赖。
