# Cypress Testing Series

这个目录存放 `Cypress 在软件测试中的应用` 系列稿件（当前为发布版初稿，可继续按站点格式做排版发布）。

## 规划主题

- 00 介绍：`00-cypress-introduction.md`
- 01 入门：`01-cypress-getting-started.md`
- 02 从 0 到 1：`02-cypress-from-0-to-1.md`
- 03 进阶：`03-cypress-advanced.md`
- 04 CI/CD：`04-cypress-in-ci-cd.md`
- 05 更多用法：`05-cypress-more-use-cases.md`

## 使用建议

- 建议先发布 00 与 01，先把“为什么用”和“怎么开始”讲清楚。
- 03 与 04 适合连发，方便把“稳定性治理”和“流水线落地”串成一条线。
- 05 更适合做系列收束篇，帮助读者从单点工具认知过渡到整体质量视角。
