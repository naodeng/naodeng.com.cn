## 系列定位

- 主题：WebdriverIO 在软件测试里的实际应用
- 状态：发布版初稿
- 来源范围：以 WebdriverIO 官方文档 `Getting Started` 为入口，延展到 `Setup Types`、`Configuration`、`Page Object Pattern`、`Github Actions`、`Component Testing`、`Visual Testing`、`Axe Core`、`Multiremote`
- 目标读者：想从零了解并逐步把 WebdriverIO 用到项目里的测试工程师、开发者和团队负责人
