# 发布前检查清单

## 内容准确性

- [ ] 工具名拼写一致（标题、正文、文件名、README）
- [ ] 关键能力点都有官方来源支撑
- [ ] 没有把推测写成事实

## 系列一致性

- [ ] 6 篇主题顺序完整
- [ ] 每篇都有导读、导航、参考资料
- [ ] 上一篇/下一篇链接正确

## 可发布性

- [ ] Frontmatter 完整且符合 schema
- [ ] 标题与 description 可用于 SEO
- [ ] 语言风格统一，避免机械重复

## 工程可用性

- [ ] 文件路径清晰且命名规范
- [ ] README 已列出完整系列
- [ ] 总目录（如 `docs/temporary/blog/README.md`）已登记新系列（如需要）
