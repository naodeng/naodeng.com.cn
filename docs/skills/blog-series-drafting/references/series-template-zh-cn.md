# 系列模板（中文）

以下模板适用于软件测试工具/框架应用类系列文章。

## 00 介绍

- 文章定位：系列名、状态、主题
- 导读：这篇回答哪三个核心问题
- 一、为什么要关注该工具
- 二、没有该工具时通常怎么做
- 三、引入后会发生什么变化
- 四、适用边界与风险提示
- 五、总结
- 系列导航：下一篇
- 参考资料：官方文档/仓库

## 01 入门

- 文章定位
- 导读：快速跑通目标
- 一、准备工作
- 二、第一个可运行示例
- 三、常见入门坑位
- 四、入门阶段最佳实践
- 五、总结
- 系列导航：上一篇/下一篇
- 参考资料

## 02 从 0 到 1

- 文章定位

- 二、最小落地方案（MVP）
- 三、团队协作机制
- 四、效果评估
- 五、总结
- 系列导航：上一篇/下一篇
- 参考资料

- 三、稳定性治理
