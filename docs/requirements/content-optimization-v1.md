# Content Optimization v1

## 目标
- 提升内容可读性、专业性与转化能力，构建“术语 -> 方法 -> 实战”的内容闭环。

## 范围
- Wiki / AIWiki 词条质量统一。
- 博客文章结构优化与内链增强。
- 中英文一致性与术语风格统一。

## 关键需求
1. 建立统一词条模板（定义、场景、最佳实践、常见误区、相关术语）。
2. 清理 markdown 格式问题（TOC、代码块、重复段落、空行）。
3. 建立术语风格表（不翻译词、缩写规则、标点规范）。
4. 每篇内容补齐 CTA（相关词条、相关文章、下一步学习路径）。
5. 为重点词条补充事实校对与来源标注。

## 成功指标
- Wiki 页面平均停留时长提升 >= 20%。
- 内容跳转率（related click）提升 >= 25%。
- 低质量页面比例下降 >= 50%。
