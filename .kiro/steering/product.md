# Product Overview

Nao's Blog (inaodeng.com) is a bilingual (English/Chinese) technical blog focused on software testing, automation, and development practices. The site features:

- Blog posts organized by categories, tags, and series
- Wiki-style testing encyclopedia (Chinese only)
- Testing automation guild/learning paths for various frameworks
- Documentation section
- Algolia-powered search
- GitHub-based comments via Giscus
- Google Analytics integration

The site is deployed globally on Cloudflare Workers for optimal performance and supports full internationalization (i18n) with Chinese (zh-cn) as default and English (en) as secondary language.

## Content Types

- **Blog**: Technical articles with tags, categories, and series support
- **Wiki**: Testing terminology encyclopedia with alphabetical organization
- **Guild**: Structured learning paths for test automation frameworks (API, UI, Performance)
- **Docs**: General documentation organized by sections
