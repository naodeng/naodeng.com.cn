# Product

Nao's Blog (inaodeng.com) is a bilingual (English / Simplified Chinese) personal blog and knowledge base focused on software testing, QA, and test automation.

## Content Areas

- Blog posts: software testing, automation, API testing, performance testing, AI-assisted testing
- QA Wiki: glossary of software testing terms (Chinese-only, 200+ entries)
- Automation Testing Guild: structured learning paths for test automation frameworks
- Docs: general documentation section

## Key Features

- Full bilingual support (en / zh-cn) with locale-prefixed URLs
- Algolia search (falls back to local JSON index)
- Giscus comments (GitHub Discussions-based)
- Google Analytics (GA4)
- RSS feeds per locale
- Deployed to Cloudflare Workers (edge)
